GetPoP Demo Install

In file getpop-demo-install.zip you will find what is needed to quickly install a copy of website GetPoP Demo in your localhost:

  • File getpop-demo-install.sql is a database dump of the website
  • Folder getpop-demo-environment is a plugin, copy it under wp-content/plugins

The website is configured to run under http://getpop-demo.localhost, and the admin credentials are:

  • Username: admin
  • Password: 11111111

Any problem with the installation, please write to info@getpop.org.
